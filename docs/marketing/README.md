# Marketing Copy – AFH Moltbot Consultant Curriculum

Ad and post copy for promoting the AFH Moltbot Consultant Curriculum across platforms.

---

## Files

| File | Platform | Use |
|------|----------|-----|
| **facebook.md** | Facebook | Feed ads, carousel, long-form engagement; targeting notes |
| **instagram.md** | Instagram | Feed posts, carousel, Stories, Reels; hashtags |
| **threads.md** | Threads | Short posts, thread replies, question hooks |
| **blogger.md** | Blogger | Full blog post (HTML-friendly paragraphs) |
| **naver-blog.md** | Naver Blog | Korean blog post + keywords |
| **tistory.md** | Tistory | Korean blog post + tags |
| **wordpress.md** | WordPress | Post title, excerpt, body (block/compatible), categories, tags |
| **newsletter.md** | Newsletter | Subject lines, preheader, body (HTML + plain text) |
| **email.md** | Email | 1:1 short, formal group, ultra-short reminder |

---

## How to Use

1. Copy the desired section from each file.
2. Replace `[링크]` / `[URL]` / `[프로젝트 URL]` with your actual project or deployment URL.
3. For Facebook/Instagram: use Meta Business Suite or Ads Manager; paste headline, primary text, and CTA.
4. For blogs: paste into Blogger / Naver / Tistory / WordPress editor; add images if needed.
5. For newsletter/email: paste subject, preheader, and body into your email tool (Mailchimp, SendGrid, etc.).

---

## Common Replacements

- **[링크]** / **[URL]** → Your portal URL (e.g. `https://your-domain.com` or GitHub repo).
- **[이름]** (in email.md) → Recipient name for 1:1 emails.
