# Quickstart – AFH Moltbot Consultant Curriculum

Get the training portal running in under 5 minutes.

---

## 1. Clone or Download

Ensure you have the project on your machine (clone the repo or download and extract).

---

## 2. Install Dependencies

```bash
cd "AFHconsultant_13(Auto Moltbot)byGe"
npm install
```

---

## 3. Start the App

```bash
npm run dev
```

You should see:

```
▲ Next.js 14.x.x
- Local: http://localhost:3000
```

---

## 4. Open in Browser

Go to **http://localhost:3000**.

- **Overview** – Course goal and how to use the portal.
- **Curriculum** – Full 4-phase curriculum and modules.
- **Moltbot Scripts** – Top 10 CLI commands.
- **Inspection Prompt Pack** – 50+ prompts.
- **Paperwork Binder** – Folder structure for AFH automation.
- **AI Playbooks** – Zero-touch scripts, NCP prompt, inspection-day workflow.
- **AI Use Boundaries** – What AI may / may not do.
- **Top 100 Questions** – Question bank and AI thinking flows.

---

## 5. Optional: Set Up Moltbot

To use the scripts and prompts with Moltbot:

```bash
npm install -g moltbot@latest
moltbot onboard --install-daemon
```

Point Moltbot’s knowledge base to your AFH folder (e.g. `~/AFH_Automation_Root`) and use the commands from the **Moltbot Scripts** and **Inspection Prompt Pack** pages.
